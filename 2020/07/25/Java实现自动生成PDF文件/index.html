一、第三方库文件

Java本身是没有能够自动生成PDF文件的API的，因此我们需要导入第三方的库文件。

这里要介绍的第三方jar包是itext，这里提供两个文件的下载：

  • itext-asian-5.2.0.jar

  • itextpdf-5.5.10.jar

二、导入到Maven

由于Maven仓库中没有提供两个文件的依赖导入方式，因此需要我们将其作为自定义的jar包导入到本地的maven仓库中。当然，如果你并没有使用依赖管理工具来管理你的依赖包，那可以直接导入到你的项目中。

  1. 我们需要执行指令 mvn install。当然，这个命令有几个参数需要介绍一下

  2. -Dfile=【文件路径】

  3. -DgroupId=【包的分组ID】

  4. -DartifactId=【包名】

  5. -Dversion=【包的版本号】

  6. -Dpackaging=【打包方式】
